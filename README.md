
# ZYNC | Adaptive Mastery & Engagement Platform (AMEP)

**ZYNC** is a futuristic EdTech operating system designed to bridge the gap between students, educators, and administrators using advanced AI. It combines adaptive learning, biometric engagement tracking, and gamified mastery paths to create a holistic educational ecosystem.

## ğŸ“¦ Installation & Setup

### 1. Prerequisites
* Node.js (v18+)
* A Google AI Studio API Key (Gemini)
* A Supabase Project

### 2. Clone and Install
```bash
git clone [https://github.com/yourusername/amep-platform.git](https://github.com/yourusername/amep-platform.git)
cd amep-platform
npm install
```
### 3. Environment Variables
Create a .env file in the root directory
(if using Vite, use .env.local):
```env
# Google Gemini API Key
API_KEY=your_google_gemini_api_key_here
```
### 4. Database Setup (Supabase)
Run the following SQL in your Supabase SQL Editor:
```sql
-- Create Profiles Table
create table public.profiles (
  id uuid references auth.users primary key,
  full_name text,
  role text default 'Student',
  created_at timestamp with time zone default now()
);

-- Auto-create profile on signup
create function public.handle_new_user()
returns trigger as $$
begin
  insert into public.profiles (id, full_name, role)
  values (
    new.id,
    new.raw_user_meta_data->>'full_name',
    new.raw_user_meta_data->>'role'
  );
  return new;
end;
$$ language plpgsql security definer;

create trigger on_auth_user_created
  after insert on auth.users
  for each row execute procedure public.handle_new_user();

-- Supporting Tables
create table public.amep_user_stats (
  user_id uuid primary key,
  xp int,
  level int,
  updated_at timestamp
);

create table public.amep_mastery (
  user_id uuid,
  subject_id text,
  score int,
  last_updated timestamp,
  primary key (user_id, subject_id)
);

create table public.amep_proctoring_logs (
  id uuid default gen_random_uuid() primary key,
  user_id uuid,
  mode text,
  report_data jsonb,
  created_at timestamp
);

create table public.amep_tasks (
  id text primary key,
  user_id uuid,
  title text,
  status text,
  priority text,
  deadline text,
  source text
);

create table public.amep_wellness_entries (
  id uuid default gen_random_uuid() primary key,
  user_id uuid,
  transcription text,
  sentiment text,
  score int,
  advice text,
  created_at timestamp
);

create table public.amep_assignments (
  teacher_id uuid,
  student_id uuid,
  primary key (teacher_id, student_id)
);
```
### 5. Run the Application
```bash
npm run dev
```
Open your browser at:
http://localhost:5173
